# SOC Lab: Detecting Successful Logins (Event ID 4624)

## Overview
This project demonstrates how to detect and analyze successful login events using Windows Event Logs and Splunk.

## Tools Used
- Windows OS
- Splunk Enterprise
- Splunk Universal Forwarder

## What I Did
- Configured Splunk to receive logs
- Installed and configured Universal Forwarder
- Collected Windows Security logs
- Detected Event ID 4624 (successful login)
- Investigated login activity

## Skills Demonstrated
- Log analysis
- SIEM (Splunk)
- Blue Team investigation
- Windows event monitoring
